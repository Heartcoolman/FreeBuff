教程

如何接入 Claude Code

按顺序操作

1. 启动桥接服务

cd /Users/liji/FreeBuff
npm start

保持这个进程运行，默认监听 `http://127.0.0.1:8765`。

2. 给 Claude Code 配环境变量

export ANTHROPIC_BASE_URL=http://127.0.0.1:8765
export ANTHROPIC_API_KEY=dummy

`ANTHROPIC_API_KEY` 这里只是占位，真正鉴权走本地 Codebuff 凭据。

3. 在 Claude Code 里使用这个模型

claude --model freebuff-bridge

当前推荐模型别名：freebuff-bridge

4. 持久化到 Claude Code 配置

可写到 `~/.claude/settings.json` 或你自己的项目级 Claude Code 配置里。

5. 第一次使用前先确认登录

如果上面的账号区域还没有可用账号，先点"新增登录账号"，浏览器完成 Codebuff 官方登录后，再点"检查登录状态"。

6. 验证是否接通

curl http://127.0.0.1:8765/health

只要 `authenticated: true`，Claude Code 就可以通过这个桥使用 Codebuff 侧能力。
